# Mobile App Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the React Native bare-workflow app (`mobile/`) that operators install on each phone to: list pending TikTok posts assigned to that device, download the video, and one-tap hand off to TikTok via the OpenSDK Share Kit. End-of-plan deliverable: a signed iOS build distributed via TestFlight + a signed Android build distributed via Google Play closed testing, with the full create-job → app-share → ack → reaction flow verified end-to-end.

**Architecture:** A standard React Native bare-workflow app with two screens (Settings and Jobs). All non-trivial logic is in plain TypeScript modules (`api/`, `auth/`, `services/`); React components stay thin. Native modules (`TikTokShareBridge` on iOS Swift + Android Kotlin) wrap the TikTok OpenSDK; everything else is JS. State is local with `AuthContext` + `useReducer`. Authentication is a per-device static bearer token stored in OS-level secure storage. Video download is lazy and foreground-only. The clipboard fallback is unconditional (toast always shown), so caption-injection quirks in TikTok's SDK never break the flow.

**Tech Stack:** React Native 0.76+ (bare workflow), TypeScript, `@react-navigation/native` + `@react-navigation/native-stack`, `react-native-keychain`, `react-native-fs`, `@react-native-clipboard/clipboard`, native iOS/Android TikTok OpenSDK. Tested with Jest + React Native Testing Library for JS; native modules verified manually on real devices (no automated harness — RN bare adds complexity not justified for this scope).

**Reference spec:** [docs/superpowers/specs/2026-04-26-mobile-tiktok-app-design.md](../specs/2026-04-26-mobile-tiktok-app-design.md). Sections 8 (Mobile App) and 9 (TikTok OpenSDK Integration) are most relevant.

**Prerequisites:**
1. Plan A (VPS server) deployed and reachable at `https://tiktok.sididi.tv`.
2. Plan B (main backend swap) merged so the live system creates real `TikTokJob`s on the VPS.
3. **TikTok Developer Portal app registration** is the first step in Task 7 — gating: no SDK code runs without it.

## Pre-Implementation Choices (locked)

These were settled during brainstorming and the spec; do not revisit:

- **Bare RN, not Expo** (TikTok OpenSDK requires custom native modules; Expo prebuild is more friction than help here).
- **Bundle / package id**: `tv.sididi.tiktokreproducer` (both platforms; matches what's whitelisted with TikTok Developer Portal).
- **Display name**: `Tiktok Reproducer`.
- **French only**, hardcoded strings — single operator, no i18n.
- **Dark mode follows system**, no toggle.
- **Single-screen one-tap** UX with **long-press for "Mark as posted"** safety valve.
- **Ack-before-share**: backend ack fires *before* the share intent so we don't depend on a return callback that doesn't reliably exist.
- **No background notifications**: Discord remains the alert; the app is foreground-only.
- **Phone-only, portrait-only**.

```typescript
import {AckResponse, ApiError, Job, MeResponse, VideoUrlResponse} from './types';

export class ApiClient {
  private readonly baseUrl: string;

  constructor(baseUrl: string, private readonly token: string) {
    this.baseUrl = baseUrl.replace(/\/$/, '');
  }

  private async request<T>(path: string, init: RequestInit = {}): Promise<T> {
    const headers: Record<string, string> = {
      Authorization: `Bearer ${this.token}`,
      ...((init.headers as Record<string, string>) ?? {}),
    };
    const response = await fetch(`${this.baseUrl}${path}`, {...init, headers});
    if (!response.ok) {
      let detail = `${response.status}`;
      try {
        const body = await response.json();
        detail = body?.detail ?? detail;
      } catch {
        /* ignore */
      }
      throw new ApiError(response.status, detail);
    }
    return (await response.json()) as T;
  }

  me(): Promise<MeResponse> {
    return this.request<MeResponse>('/api/mobile/me');
  }

  listJobs(): Promise<Job[]> {
    return this.request<Job[]>('/api/mobile/jobs');
  }

  videoUrl(jobId: string): Promise<VideoUrlResponse> {
    return this.request<VideoUrlResponse>(`/api/mobile/jobs/${jobId}/video-url`);
  }

  ack(jobId: string): Promise<AckResponse> {
    return this.request<AckResponse>(`/api/mobile/jobs/${jobId}/ack`, {method: 'POST'});
  }
}
```

```typescript
import * as Keychain from 'react-native-keychain';

const SERVICE = 'tv.sididi.tiktokreproducer.api';

export interface Credentials {
  backendUrl: string;
  token: string;
}

export async function saveCredentials(backendUrl: string, token: string): Promise<void> {
  await Keychain.setGenericPassword(backendUrl, token, {
    service: SERVICE,
    accessible: Keychain.ACCESSIBLE.WHEN_UNLOCKED,
  });
}

export async function loadCredentials(): Promise<Credentials | null> {
  const result = await Keychain.getGenericPassword({service: SERVICE});
  if (!result) {
    return null;
  }
  return {backendUrl: result.username, token: result.password};
}

export async function clearCredentials(): Promise<void> {
  await Keychain.resetGenericPassword({service: SERVICE});
}
```

```tsx
import React, {createContext, useCallback, useContext, useEffect, useState} from 'react';
import {ApiClient} from '../api/client';
import {AccountSummary} from '../api/types';
import {Credentials, loadCredentials, saveCredentials, clearCredentials} from './secureStore';

interface AuthState {
  credentials: Credentials | null;
  deviceId: string | null;
  accounts: AccountSummary[];
  status: 'loading' | 'unauthenticated' | 'authenticated';
}

interface AuthContextValue extends AuthState {
  client: ApiClient | null;
  signIn: (backendUrl: string, token: string) => Promise<void>;
  signOut: () => Promise<void>;
}

const AuthContext = createContext<AuthContextValue | null>(null);

export const useAuth = (): AuthContextValue => {
  const ctx = useContext(AuthContext);
  if (!ctx) throw new Error('useAuth must be used inside AuthProvider');
  return ctx;
};

export const AuthProvider: React.FC<{children: React.ReactNode}> = ({children}) => {
  const [state, setState] = useState<AuthState>({
    credentials: null,
    deviceId: null,
    accounts: [],
    status: 'loading',
  });

  const verify = useCallback(async (creds: Credentials): Promise<AuthState> => {
    const c = new ApiClient(creds.backendUrl, creds.token);
    const me = await c.me();
    return {
      credentials: creds,
      deviceId: me.device_id,
      accounts: me.accounts,
      status: 'authenticated',
    };
  }, []);

  useEffect(() => {
    void (async () => {
      const stored = await loadCredentials();
      if (!stored) {
        setState(s => ({...s, status: 'unauthenticated'}));
        return;
      }
      try {
        setState(await verify(stored));
      } catch {
        await clearCredentials();
        setState({credentials: null, deviceId: null, accounts: [], status: 'unauthenticated'});
      }
    })();
  }, [verify]);

  const signIn = useCallback(
    async (backendUrl: string, token: string): Promise<void> => {
      const next = await verify({backendUrl, token});
      await saveCredentials(backendUrl, token);
      setState(next);
    },
    [verify],
  );

  const signOut = useCallback(async () => {
    await clearCredentials();
    setState({credentials: null, deviceId: null, accounts: [], status: 'unauthenticated'});
  }, []);

  const client =
    state.credentials && state.status === 'authenticated'
      ? new ApiClient(state.credentials.backendUrl, state.credentials.token)
      : null;

  return (
    <AuthContext.Provider value={{...state, client, signIn, signOut}}>
      {children}
    </AuthContext.Provider>
  );
};
```

```typescript
import RNFS from 'react-native-fs';

export const localPathFor = (jobId: string): string =>
  `${RNFS.CachesDirectoryPath}/${jobId}.mp4`;

export type ProgressCallback = (bytesWritten: number, contentLength: number) => void;

export async function downloadVideo(
  url: string,
  jobId: string,
  onProgress: ProgressCallback,
): Promise<string> {
  const dest = localPathFor(jobId);
  if (await RNFS.exists(dest)) {
    await RNFS.unlink(dest);
  }
  const {promise} = RNFS.downloadFile({
    fromUrl: url,
    toFile: dest,
    progressInterval: 250,
    progress: ({bytesWritten, contentLength}) =>
      onProgress(bytesWritten, contentLength || 1),
  });
  const result = await promise;
  if (result.statusCode < 200 || result.statusCode >= 300) {
    throw new Error(`Download failed with status ${result.statusCode}`);
  }
  return dest;
}
```

```typescript
  const out: string[] = [];
```

```tsx
const showToast = (msg: string) => {
  if (Platform.OS === 'android') {
    ToastAndroid.show(msg, ToastAndroid.SHORT);
  } else {
    Alert.alert(msg);
  }

export const JobsScreen: React.FC = () => {
  const {client} = useAuth();
  const [jobs, setJobs] = useState<Job[]>([]);
  const [refreshing, setRefreshing] = useState(false);
  const [loading, setLoading] = useState(true);
  const [cardStates, setCardStates] = useState<Record<string, JobCardState>>({});

  const setCardState = useCallback((jobId: string, state: JobCardState) => {
    setCardStates(prev => ({...prev, [jobId]: state}));
  }, []);

  const refresh = useCallback(async () => {
    if (!client) return;
    setRefreshing(true);
    try {
      const data = await client.listJobs();
      setJobs(data);
    } catch (e) {
      Alert.alert('Erreur', String((e as Error).message ?? e));
    } finally {
      setRefreshing(false);
      setLoading(false);
    }
  }, [client]);

  useEffect(() => {
    void refresh();
  }, [refresh]);

  useEffect(() => {
    const sub = AppState.addEventListener('change', s => {
      if (s === 'active') void refresh();
    });
    return () => sub.remove();
  }, [refresh]);

  const handlePress = useCallback(
    async (job: Job) => {
      if (!client) return;
      setCardState(job.job_id, {kind: 'fetchingUrl'});
      try {
        const {video_url} = await client.videoUrl(job.job_id);
        setCardState(job.job_id, {kind: 'downloading', progress: 0});
        const localPath = await downloadVideo(video_url, job.job_id, (b, t) =>
          setCardState(job.job_id, {kind: 'downloading', progress: b / t}),
        );
        setCardState(job.job_id, {kind: 'sharing'});
        copyToClipboard(job.description);
        await client.ack(job.job_id);
        await shareToTikTok({
          videoPath: localPath,
          hashtags: extractHashtags(job.description),
        });
        showToast(TOAST_TEXT);
        // optimistic removal: card disappears
        setJobs(prev => prev.filter(j => j.job_id !== job.job_id));
      } catch (e) {
        const message = (e as Error).message ?? String(e);
        setCardState(job.job_id, {kind: 'error', message});
      }
    },
    [client, setCardState],
  );

    (job: Job) => {
      Alert.alert(

                await client.ack(job.job_id);
                setJobs(prev => prev.filter(j => j.job_id !== job.job_id));
              } catch (e) {
                Alert.alert('Erreur', String((e as Error).message ?? e));
              }
            },
          },
        ],
      );
    },
    [client],

  );
};
```

```swift
import Foundation
import React
import TikTokOpenSDKShare

@objc(TikTokShareBridge)
class TikTokShareBridge: NSObject {

  @objc static func requiresMainQueueSetup() -> Bool { return true }

  @objc func share(_ params: NSDictionary,
                   resolver resolve: @escaping RCTPromiseResolveBlock,
                   rejecter reject: @escaping RCTPromiseRejectBlock) {
    guard let videoPath = params["videoPath"] as? String else {
      reject("bad_params", "videoPath required", nil); return
    }
    let hashtags = params["hashtags"] as? [String] ?? []

    let req = TikTokShareRequest()
    req.mediaType = .video
    req.localIdentifiers = [videoPath]
    req.hashtagNames = hashtags

    DispatchQueue.main.async {
      req.send { response in
        if let error = response.error {
          reject("share_failed", error.localizedDescription, error)
        } else {
          resolve(nil)
        }
      }
    }
  }
}
```

```kotlin
  fun share(params: ReadableMap, promise: Promise) {
    val videoPath = params.getString("videoPath")
      ?: return promise.reject("bad_params", "videoPath required")
    val hashtags = params.getArray("hashtags")
      ?.toArrayList()
      ?.map { it.toString() }
      ?: emptyList()
    val activity = currentActivity
      ?: return promise.reject("no_activity", "No activity")

    try {
      val mediaContent = MediaContent(
        mediaType = MediaType.VIDEO,
        mediaPaths = arrayListOf(videoPath)
      )
      val request = Share.Request(
        mediaContent = mediaContent,
        packageName = activity.packageName,
        resultActivityFullPath = "${activity.packageName}.TikTokEntryActivity",
        hashtagList = ArrayList(hashtags)
      )
      ShareApi(activity).share(request)
      promise.resolve(null)
    } catch (e: Exception) {
      promise.reject("share_failed", e.message, e)
    }
  }
}
```
